 

1.What is the best-case time complexity of Quick Sort?**  
   a) O(n²)  
   b) O(n log n)  
   c) O(n)  
   d) O(log n) 




















 
   **Answer:** b) O(n log n)  

2. **Binary Search requires the input array to be:**  
   a) Sorted  
   b) Unsorted  
   c) In reverse order  
   d) None of the above 











 
   **Answer:** a) Sorted  

3. **Which sorting algorithm works by repeatedly swapping adjacent elements if they are in the wrong order?**  
   a) Merge Sort  
   b) Quick Sort  
   c) Bubble Sort  
   d) Radix Sort  














   **Answer:** c) Bubble Sort  

4. **In Heapsort, which data structure is used to implement the algorithm?**  
   a) Stack  
   b) Queue  
   c) Heap  
   d) Graph 














 
   **Answer:** c) Heap  

5. **Which algorithm is used for finding the shortest path in an unweighted graph?**

Here are 30 multiple-choice questions (MCQs) on **Foundations of Algorithm** with answers, ranging from high to low difficulty.  
  

1. **Which of the following problems is undecidable?**  
   a) 3-SAT  
   b) Traveling Salesman Problem  
   c) Halting Problem  
   d) Longest Common Subsequence 











 
   **Answer:** c) Halting Problem  

2. **If an algorithm has time complexity O(2^n), which of the following statements is true?**  
   a) It is a polynomial-time algorithm.  
   b) It is an exponential-time algorithm.  
   c) It has constant time complexity.  
   d) It is a logarithmic-time algorithm. 








 
   **Answer:** b) It is an exponential-time algorithm.  

3. **Which data structure is used in the implementation of recursion?**  
   a) Queue  
   b) Stack  
   c) Heap  
   d) Graph  













   **Answer:** b) Stack  

4. **Which sorting algorithm has an average-case time complexity of O(n log n) and is not in-place?**  
   a) Quick Sort  
   b) Merge Sort  
   c) Heap Sort  
   d) Bubble Sort 








 
   **Answer:** b) Merge Sort  

5. **The Master Theorem is used for solving recurrence relations of the form:**  
   a) T(n) = aT(n/b) + O(n^c)  
   b) T(n) = T(n-1) + O(n)  
   c) T(n) = T(n/2) + O(log n)  
   d) T(n) = O(1) 








 
   **Answer:** a) T(n) = aT(n/b) + O(n^c)  

6. **Which of the following best describes an NP-complete problem?**  
   a) Can be solved in polynomial time.  
   b) Can be reduced to an NP problem in polynomial time.  
   c) Can be solved in exponential time only.  
   d) Cannot be solved by any algorithm.










  
   **Answer:** b) Can be reduced to an NP problem in polynomial time.

   **Answer:** b) Dynamic Programming

   **Answer:** c) Produces different outputs for the same input
